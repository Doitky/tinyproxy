# tinyproxy

Tinyproxy is a lightweight HTTP/HTTPS proxy daemon for POSIX operating systems. It is designed to be small and efficient, and is ideal for use in embedded systems and resource-constrained environments.

## Features
- Lightweight and fast
- Easy to configure
- Supports HTTP and HTTPS proxying
- Designed for small resource usage
